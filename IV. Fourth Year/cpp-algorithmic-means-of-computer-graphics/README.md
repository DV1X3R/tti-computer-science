```
–ö–æ–º–ø–æ–∑–∏—Ü–∏—è 2D –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–π
V* = T* * R* * S*
P*2 = P*1 * V*

–ü–æ–≤–æ—Ä–æ—Ç —Ç–æ—á–∫–∏ –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ —Ç–æ—á–∫–∏
–ü–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –±–∞–∑–æ–≤—ã—Ö –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–π:
1. –ü–µ—Ä–µ–Ω–æ—Å –°–ö –≤ —Ç–æ—á–∫—É P0(X0,Y0) -> T*^-1(X0,Y0)
2. –ü–æ–≤–æ—Ä–æ—Ç —Ç–æ—á–∫–∏ P1 –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –Ω–∞—á–∞–ª–∞ –°–ö –Ω–∞ —É–≥–æ–ª Œ∏ -> R*(Œ∏)
3. –ü–µ—Ä–µ–Ω–æ—Å –°–ö –æ–±—Ä–∞—Ç–Ω–æ -> T*^-1(-X0,-Y0)
    P*2 = P*1 * T*^-1(X0,Y0) * R*(Œ∏) * T*^-1(-X0,-Y0)
    P*2 = P*1 * R*0, –≥–¥–µ
    R*0 = T*^-1(X0,Y0) * R*(Œ∏) * T*^-1(-X0,-Y0)

–ú–∞—Å—à—Ç–∞–±–∏—Ä–æ–≤–∞–Ω–∏–µ —Ç–æ—á–∫–∏ –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ —Ç–æ—á–∫–∏
–ü–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –±–∞–∑–æ–≤—ã—Ö –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–π:
1. –ü–µ—Ä–µ–Ω–æ—Å –°–ö –≤ —Ç–æ—á–∫—É P0(X0,Y0) -> T*^-1(X0,Y0)
2. –ú–∞—Å—à—Ç–∞–±–∏—Ä–æ–≤–∞–Ω–∏–µ P1 –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –Ω–∞—á–∞–ª–∞ –°–ö -> S*(Sx,Sy)
3. –ü–µ—Ä–µ–Ω–æ—Å –°–ö –æ–±—Ä–∞—Ç–Ω–æ -> T*^-1(-X0,-Y0)
    P*2 = P*1 * T*^-1(X0,Y0) * S*(Sx,Sy) * T*^-1(-X0,-Y0)
	P*2 = P*1 * S*0, –≥–¥–µ
	S*0 = T*^-1(X0,Y0) * S*(Sx,Sy) * T*^-1(-X0,-Y0)
```

```python
import numpy as np
import math as m

def t(Dx, Dy):
    return [[1, 0, 0], [0, 1, 0], [Dx, Dy, 1]]

def s(Sx, Sy):
    return [[Sx, 0, 0], [0, Sy, 0], [0, 0, 1]]

def r(O):
    o = m.radians(O)
    return [[m.cos(o), m.sin(o), 0], [-m.sin(o), m.cos(o), 0], [0, 0, 1]]

def sr(src, X0, Y0, Sx, Sy, O):
    p = np.matmul(t(-X0, -Y0), s(Sx, Sy))
    p = np.matmul(p, r(O))
    p = np.matmul(p, t(X0, Y0))
    return np.matmul(src, p)

a = [1, 2, 1]
b = [1, 4, 1]
c = [2, 2, 1]

X0 = 1; Y0 = 2
Sx = 2; Sy = 1
O = -90

print('a', a)
print('b', b)
print('c', c)
print('a2', sr(a, X0, Y0, Sx, Sy, O)) # [1, 2, 1]
print('b2', sr(b, X0, Y0, Sx, Sy, O)) # [3, 2, 1]
print('c2', sr(c, X0, Y0, Sx, Sy, O)) # [1, 0, 1]
```
